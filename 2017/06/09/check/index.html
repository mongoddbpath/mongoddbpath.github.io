使用Hexo遇到的Bug以及修复

发表于 2017-06-09 | 分类于 前端 |  阅读次数 次

Local hexo not found in的解决方法…


  小编遇到这个问题两次了，第一次解决Bug花了一些功夫去修复的，第二次就很快，这是个不常见错误，就是当你用npm下载或者卸载一些插件或者包的时候，原有的node_modules里面就会有一定可能性丢失一些原有的包，一个框架是依赖于包与包的之间的关系的，所以就出现了这样的错误，Local即当前目录的Hexo指令找不到失效了…
  这时候不管重新下载hexo或者是安装不同的Nodejs的版本都是没用的，解决的方法是:
  建立一个新的文件夹Hexo-CheckOut，执行hexo init初始化命令，找到新建的文件夹Hexo-CheckOut下的node_modules文件夹，把命令行中提示的缺少的包复制到原来博客的node_modules文件夹下即可…

cd ..
mkdir Hexo-CheckOut
hexo init



  于是乎，命令又能重新启动了~
