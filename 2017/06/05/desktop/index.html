一个程序员的电脑桌面

发表于 2017-06-05 | 分类于 私房菜 |  阅读次数 次

你永远无法想象一个程序员的桌面到底有多整洁…

  相信大多数用Windows的帅哥美女的电脑桌面都像下面这张图一样，桌面充满了大量的应用的快捷方式；

  过多的软件快捷方式会使得桌面过乱不易管理，有时候还影响美观，与众不同，我的电脑桌面却是下面这个模样…

  What！！你确定不是一张图片嘛，什么快捷键都没有，那你平常都怎么打开应用，而且那四个白色图标是什么鬼？？别急，听小编慢慢道来…
  2017年开学的时候，大概三月份左右，一个突如其来的想法闪了我的大脑，我是IT专业的，为什么我的电脑桌面会和别人一样，不，我要不同！！！而这时，一个实验敲中了我的火花，依稀记得做Java实验的时候说过windows的Path变量是可以指定可执行文件的路径的，那是不是意味着我可以把快捷键隐藏，用cmder(一个替换Windows命令窗口cmd的工具)通过Path变量就找到可执行文件的目录进而打开软件？？秉着“生命不息，折腾不止”的精神，经过一段时间的捣鼓，一个下午的时间，我就把电脑桌面改成了这样，至于怎么配的…

步骤如下：
  首先我们下载软件一般在目录下有个.exe文件，那是Windows下的可执行文件，而如果Windows通过Path变量或者系统变量找到了他，就可能通过命令行来启动，所以我们先将该可执行文件目录找到，以微信为例

  其次，我们得把该地址拷贝下来，找到我的电脑->属性->高级系统设置->环境变量->系统变量(Path),选择编辑，新建…




  最后，我们把该地址添加到Path变量上，于是通过win + r 调出 cmd ， 然后通过命令(也就是该exe文件的名字，[exe文件名可以自己设置])打开软件，这样桌面的快捷方式就显得不必要了！！


  当然，并不是所有都能成功，不过大部分的exe文件都可以，如果path变量不行，那就把.exe文件目录拷到系统变量(新建系统变量，拷贝exe文件地址)下，通过start + exe文件名就可以运行了

  至于图标为什么是白色的，很简单的，下载一套白色图标，右击你要选择软件的可执行文件.exe的属性，找到替换图标即可，需要的同学我已经把白色图标上传到了百度云，地址是：http://pan.baidu.com/s/1nvfX6uX，密码是：yzm7
  对于精简主义的我来说，这样的桌面是大爱，没有冗余，简洁大方，Cool~
  当然普通用户倒是没必要啦，毕竟双击貌似也是不错的选择呢~
