一个程序员的电脑桌面

发表于 2017-06-05 | 分类于 私房菜 |  阅读次数 次

一个程序员的桌面到底有多整洁…


  相信大多数用Windows的帅哥美女都像下面这张图一样，桌面充满了大量的应用的快捷方式；
,过多的软件快捷方式会使得桌面过乱不易管理，有时候还影响美观，与众不同，我的电脑却是下面这个模样…

  What！！你确定不是一张图片嘛，什么快捷键都没有，那你平常都怎么打开应用，而且那三个白色图标是什么鬼？？别急，听小编慢慢道来…
  2017年开学的时候，大概三月份左右，一个突如其来的想法响彻在我的大脑里，我是IT专业的，为什么我的电脑桌面会和别人一样，我要不同！！！而这时，一个实验敲中了我的火花，windows的Path变量是可以指定可执行文件的路径的，那是不是意味着我可以把快捷键隐藏，用cmder(一个替换Windows命令窗口cmd的工具)打开软件？？秉着“生命不息，折腾不止”的精神，果然行得通…

步骤如下：

首先我们下载软件一般在目录下有个.exe文件，那是Windows下的可执行文件，而如果Windows通过Path变量找到了他，就可能通过命令行来启动，所以我们先将该可执行文件找到，以微信为例

其次，我们得把该地址拷贝下来，找到我的电脑->属性->高级系统设置->环境变量->系统变量(Path),选择编辑…
